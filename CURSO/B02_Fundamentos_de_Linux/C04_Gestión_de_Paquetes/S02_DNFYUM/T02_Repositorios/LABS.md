# Labs — Repositorios

## Descripcion

Laboratorio practico para explorar los repositorios en RHEL/AlmaLinux:
archivos .repo, BaseOS vs AppStream, EPEL y CRB, claves GPG, y
gestion de repos con dnf config-manager.

## Prerequisitos

- Entorno del curso levantado (alma-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Archivos .repo | Formato INI, campos, variables |
| 2 | BaseOS, AppStream, EPEL | Repos estandar, habilitar extras |
| 3 | GPG y gestion de repos | Claves, config-manager, COPR |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```
