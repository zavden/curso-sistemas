# Labs — Operaciones

## Descripcion

Laboratorio practico para dominar las operaciones de dnf: instalar,
buscar, provides, grupos de paquetes, module streams, e historial
de transacciones con undo/redo.

## Prerequisitos

- Entorno del curso levantado (alma-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Operaciones basicas | install, remove, search, provides, list |
| 2 | Grupos y modulos | group install/info, module streams |
| 3 | Historial | dnf history, info, undo, cache |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```

Seguir las instrucciones de `labs/README.md` parte por parte.

## Tiempo estimado

~20 minutos.
