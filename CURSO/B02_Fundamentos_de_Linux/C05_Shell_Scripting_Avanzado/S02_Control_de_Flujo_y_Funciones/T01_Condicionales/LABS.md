# Labs — Condicionales

## Descripcion

Laboratorio practico para dominar las condicionales en bash:
test/[]/[[]], file tests, comparaciones de strings y numeros,
(( )) aritmetico, if/elif/else, y case con patrones avanzados.

## Prerequisitos

- Entorno del curso levantado (debian-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | File tests y comparaciones | -f, -d, -L, -r, -w, -x, -s, -nt, -ot |
| 2 | Strings y numeros | -z, -n, ==, =~, -eq, -lt, (( )) |
| 3 | case | Patrones, ;& fall-through, ;;& continue |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```

Seguir las instrucciones de `labs/README.md` parte por parte.
