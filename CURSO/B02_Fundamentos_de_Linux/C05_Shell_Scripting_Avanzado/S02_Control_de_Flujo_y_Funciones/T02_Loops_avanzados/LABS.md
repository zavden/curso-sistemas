# Labs — Loops avanzados

## Descripcion

Laboratorio practico para dominar loops en bash: for clasico,
brace expansion, C-style for, while read con IFS, process
substitution <() >(), until, break/continue N, find -print0,
y globstar.

## Prerequisitos

- Entorno del curso levantado (debian-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Variantes de for | Clasico, brace expansion, C-style, arrays |
| 2 | while read y process substitution | IFS= -r, pipeline subshell, <() |
| 3 | Control de flujo | break/continue N, find -print0, globstar |

## Archivos

labs/
