# Labs — Getopts

## Descripcion

Laboratorio practico para dominar el parsing de argumentos en
bash: getopts POSIX (OPTSTRING, OPTARG, OPTIND), modo silencioso,
shift, opciones largas con while/case manual, validacion de input,
y el patron profesional completo.

## Prerequisitos

- Entorno del curso levantado (debian-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | getopts POSIX | OPTSTRING, OPTARG, OPTIND, modo silencioso |
| 2 | shift y opciones largas | shift, while/case manual, -- |
| 3 | Validacion y patron completo | Input validation, script profesional |

## Archivos
