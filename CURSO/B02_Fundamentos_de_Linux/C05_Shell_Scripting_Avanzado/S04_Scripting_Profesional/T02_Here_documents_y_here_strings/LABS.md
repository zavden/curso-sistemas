# Labs — Here documents y here strings

## Descripcion

Laboratorio practico para dominar here documents (<< con y sin
expansion, <<-), here strings (<<<), usos comunes (crear archivos,
generar scripts, mensajes de uso), asignar a variables, y
comparacion entre las formas.

## Prerequisitos

- Entorno del curso levantado (debian-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Here documents | << con/sin expansion, <<- con tabs |
| 2 | Usos comunes | Crear archivos, generar scripts, usage |
| 3 | Here strings | <<<, parsing, comparacion con pipes |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```

Seguir las instrucciones de `labs/README.md` parte por parte.

~15 minutos.
