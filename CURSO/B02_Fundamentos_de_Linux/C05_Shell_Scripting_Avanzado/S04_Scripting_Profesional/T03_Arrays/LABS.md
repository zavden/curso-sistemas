# Labs — Arrays

## Descripcion

Laboratorio practico para dominar arrays en bash: arrays indexados
(crear, acceder, [@] vs [*], modificar, slicing), arrays asociativos
(declare -A, claves, -v), mapfile/readarray, pasar a funciones,
nameref, y patrones comunes (stack, join/split, acumular).

## Prerequisitos

- Entorno del curso levantado (debian-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Arrays indexados | Crear, acceder, [@] vs [*], modificar, slice |
| 2 | Arrays asociativos | declare -A, claves, -v, conteo, config |
| 3 | mapfile y patrones | readarray, funciones, join/split, stack |

## Archivos

```
labs/

```

## Como ejecutar
