# Labs — Contraseñas

## Descripcion

Laboratorio practico para gestionar contrasenas: passwd, chage,
algoritmos de hash, politicas de expiracion, y la diferencia entre
bloquear contrasena y bloquear cuenta.

## Prerequisitos

- Entorno del curso levantado (debian-dev y alma-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | passwd y estado | Establecer, bloquear, ver estado |
| 2 | chage y politicas | Expiracion, dias minimos/maximos, forzar cambio |
| 3 | Algoritmos y comparacion | Prefijos de hash, Debian vs AlmaLinux |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```

Seguir las instrucciones de `labs/README.md` parte por parte.

~15 minutos.
