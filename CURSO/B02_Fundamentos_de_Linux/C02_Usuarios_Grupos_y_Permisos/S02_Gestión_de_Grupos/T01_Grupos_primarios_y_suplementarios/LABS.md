# Labs — Grupos primarios y suplementarios

## Descripcion

Laboratorio practico para entender la diferencia entre grupo primario
y suplementarios: como afectan la creacion de archivos, como se
verifican los permisos, y como gestionar las membresías.

## Prerequisitos

- Entorno del curso levantado (debian-dev y alma-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Inspeccionar grupos | Primario vs suplementarios, id, groups |
| 2 | Efecto en archivos | Grupo propietario al crear archivos |
| 3 | Permisos y membresía | Verificacion de permisos, agregar/quitar |

## Archivos

```
labs/
```
