# Labs — Diferencias entre distribuciones

## Descripcion

Laboratorio practico para comparar Debian y AlmaLinux lado a lado:
identificacion del sistema, logs, gestion de paquetes, nombres de
servicios, y como escribir scripts portables entre ambas familias.

## Prerequisitos

- Entorno del curso levantado (debian-dev y alma-dev corriendo)
- Estar en el directorio `labs/` de este topico

## Contenido del laboratorio

| Parte | Concepto | Que se demuestra |
|---|---|---|
| 1 | Identificar la distribucion | os-release, archivos especificos |
| 2 | Comparar archivos y rutas | Logs, paquetes, configuracion |
| 3 | Scripts portables | Deteccion de familia, paths condicionales |

## Archivos

```
labs/
└── README.md    ← Guia paso a paso (documento principal del lab)
```

## Como ejecutar

```bash
cd labs/
```

Seguir las instrucciones de `labs/README.md` parte por parte.

~15 minutos.
